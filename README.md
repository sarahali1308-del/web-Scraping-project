Web Scraping: World's Largest Companies by Revenue
This project was completed as part of my Data Analytics Internship. It demonstrates the ability to extract, clean, and structure data from the web for business intelligence.

🛠️ Tools & Technologies

Python: Core programming language.

Pandas: For data cleaning and structuring the final table.

BeautifulSoup/Requests: For parsing HTML from Wikipedia.

Jupyter Notebook: For interactive development and testing.

📊 Key Features

Scraped real-time data from Wikipedia's list of largest companies.

Cleaned "dirty" web data (removing citations like [1] and formatting numbers).

Exported a structured CSV file ready for Excel analysis.

📁 Files in this Repo

web scraping.ipynb: The full Python script and analysis.

largest_companies.csv: The final cleaned dataset.
